Reporte 3: Problema 2

Introduccion

Se implementara un FIS de segundo tipo, reproduciendo la implementacion de la pagina oficial de MATLAB.

FIS de tipo uno

en este apartado, se crea el FIS de tipo uno que servira como base para crear el FIS tipo 2
fis1 = sugfis;
se le agregan las variables de entrada al sistema:
fis1 = addInput(fis1,[-1 1],'Name','E');
fis1 = addInput(fis1,[-1 1],'Name','delE');
se toman tres funciones de membresia para cada entrada, los nombres son negativo(N), cero (Z) y positivo (P)
fis1 = addMF(fis1,'E','trimf',[-2 -1 0],'Name','N');
fis1 = addMF(fis1,'E','trimf',[-1 0 1],'Name','Z');
fis1 = addMF(fis1,'E','trimf',[0 1 2],'Name','P');
fis1 = addMF(fis1,'delE','trimf',[-2 -1 0],'Name','N');
fis1 = addMF(fis1,'delE','trimf',[-1 0 1],'Name','Z');
fis1 = addMF(fis1,'delE','trimf',[0 1 2],'Name','P');
se muestran las funciones de membresia
figure
subplot(1,2,1)
plotmf(fis1,'input',1)
title('Input 1')
subplot(1,2,2)
plotmf(fis1,'input',2)
title('Input 2')
Se agrega la variable de salida al FIS:
fis1 = addOutput(fis1,[-1 1],'Name','U');
se agregan funciones constantes la salida, los nombres de estas son gran negativo(NB), medio negativo(NM), cero(Z), Medio positivo(PM) y gran positivo(PB).
fis1 = addMF(fis1,'U','constant',-1,'Name','NB');
fis1 = addMF(fis1,'U','constant',-0.5,'Name','NM');
fis1 = addMF(fis1,'U','constant',0,'Name','Z');
fis1 = addMF(fis1,'U','constant',0.5,'Name','PM');
fis1 = addMF(fis1,'U','constant',1,'Name','PB');
se agregan las reglas al FIS que crean una superficie de control proporcional:
rules = [...
"E==N & delE==N => U=NB"; ...
"E==Z & delE==N => U=NM"; ...
"E==P & delE==N => U=Z"; ...
"E==N & delE==Z => U=NM"; ...
"E==Z & delE==Z => U=Z"; ...
"E==P & delE==Z => U=PM"; ...
"E==N & delE==P => U=Z"; ...
"E==Z & delE==P => U=PM"; ...
"E==P & delE==P => U=PB" ...
];
fis1 = addRule(fis1,rules);
Se muestra la figura:
figure
gensurf(fis1)
title('Control surface of type-1 FIS')

FIS de tipo 2

Se convierte el FIS de tipo 1 en un FIS de tipo 2
fis2 = convertToType2(fis1);
se elige un factor de escalamiento para crear el rango entre las dos funciones de membresia que conforman los conjuntos tipo 2
scale = [0.2 0.9 0.2;0.3 0.9 0.3];
for i = 1:length(fis2.Inputs)
for j = 1:length(fis2.Inputs(i).MembershipFunctions)
fis2.Inputs(i).MembershipFunctions(j).LowerLag = 0;
fis2.Inputs(i).MembershipFunctions(j).LowerScale = scale(i,j);
end
end
se muestran las dos entradas del FIS de segundo tipo
figure
subplot(1,2,1)
plotmf(fis2,'input',1)
title('Input 1')
subplot(1,2,2)
plotmf(fis2,'input',2)
title('Input 2')
El patron de incertidumbre genera una superficie no lineal de control
figure
gensurf(fis2)
title('Control surface of type-2 FIS')
